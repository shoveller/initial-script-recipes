# prj.sh
> 웹 프론트엔드 프로젝트 스케폴드 자동화 스크립트(osx 전용)

## 다운로드
```shell
curl -L -o prj.sh https://github.com/shoveller/initial-script-recipes/raw/main/prj.sh
```

## 실행권한 부여
```shell
chmod +x prj.sh
```

## 웹 프론트엔드 프로젝트 스케폴드 개시
```shell
./prj.sh
```
